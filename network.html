Chapter 4 Terms

TERM DEFINITION
Network Classifications
  • LAN
  • MAN
  • WAN
  • A computer network that interconnects computers within a limited area such as a residence, school, laboratory, university campus or office building.
  • A computer network that interconnects users with computer resources in a geographic region of the size of a metropolitan area.
  • A telecommunications network that extends over a large geographic area.
Network Models
  • Bus Network
  • Star Network
  • A network topology in which nodes are directly connected to a common half-duplex link called a bus.
  • An implementation of a spoke-hub distribution paradigm in computer networks.
Protocols
  • CSMA/CD Ethernet
  • CSMA/CD Wifi
  • A medium access control method used most notably in early Ethernet technology for local area networking.
  • A network multiple access method in which carrier sensing is used, but nodes attempt to avoid collisions by beginning transmission only after the channel is sensed to be "idle".
Combining Networks:
  • Repeater
  • Bridge
  • Switch
  • Router
  • Gateway
  • A repeater is an electronic device that receives a signal and retransmits it.
  • A computer networking device that creates a single, aggregate network from multiple communication networks or network segments.
  • Networking hardware that connects devices on a computer network by using packet switching to receive and forward data to the destination device.
  • A networking device that forwards data packets between computer networks.
  • A piece of networking hardware or software used in telecommunications networks that allows data to flow from one discrete network to another.
Client/Server Model
  • P2P: peer-to-peer
  • Distributed System
  • Cloud Computing
  • A distributed application architecture that partitions tasks or workloads between peers.
  • A field of computer science that studies distributed systems.
  • The on-demand availability of computer system resources, especially data storage and computing power, without direct active management by the user.
Internet Addressing
  • Tier 1
  • Tier 2
  • Tier 3
  • An Internet Protocol (IP) network that can reach every other network on the Internet solely via settlement-free interconnection (also known as settlement-free peering).
  • A service provider that utilizes a combination of paid transit via Tier 1 ISPs and peering with other Tier 2 ISPs to deliver Internet traffic to end customers through Tier 3 ISPs.
  • Primarily engaged in delivering Internet access to end customers.
IP Address
  • ICANN
  • DNS
  • TLD: Top-level Domain
  • Country-code TLDs
  • An American multistakeholder group and nonprofit organization responsible for coordinating the maintenance and procedures of several databases related to the namespaces and numerical spaces of the Internet, ensuring the network's stable and secure operation.
  • A hierarchical and distributed naming system for computers, services, and other resources in the Internet or other Internet Protocol networks.
  • One of the domains at the highest level in the hierarchical Domain Name System of the Internet after the root domain.
  • An Internet top-level domain generally used or reserved for a country, sovereign state, or dependent territory identified with a country code.
Internet Protocols
  • FTP
  • HTTP
  • SMTP
  • A standard communication protocol used for the transfer of computer files from a server to a client on a computer network.
  • An application layer protocol in the Internet protocol suite model for distributed, collaborative, hypermedia information systems.
  • An Internet standard communication protocol for electronic mail transmission.
World Wide Wed
  • URL
  • Client-side Activities
  • Server-side Activities
  • A reference to a web resource that specifies its location on a computer network and a mechanism for retrieving it.
  • Everything in a web application that is displayed or takes place on the client (end user device).
  • Everything that happens on the server, instead of on the client.
Internet Four-layer Software
  • Application layer
  • Transport Layer
  • Network Layer
  • Link Layer
  • An abstraction layer that specifies the shared communications protocols and interface methods used by hosts in a communications network.
  • A conceptual division of methods in the layered architecture of protocols in the network stack in the Internet protocol suite and the OSI model.
  • Responsible for packet forwarding including routing through intermediate routers.
  • The group of methods and communications protocols confined to the link that a host is physically connected to.
Internet is:
  • Packet-switching Network
  • Fault Tolerant
  • Redundant
  • A method of grouping data into packets that are transmitted over a digital network.
  • The property that enables a system to continue operating properly in the event of the failure of one or more faults within some of its components.
  • A communications pathway that has additional links to connect all nodes in case one link goes down.
Cybersecurity
  • Malware
  • Virus
  • Worm
  • Trojan Horse
  • Spyware
  • Phishing
  • DDoS Attack
  • Software that is specifically designed to disrupt, damage, or gain unauthorized access to a computer system.
  • A piece of code which is capable of copying itself and typically has a detrimental effect, such as corrupting the system or destroying data.
  • A self-replicating program able to propagate itself across a network, typically having a detrimental effect.
  • A program designed to breach the security of a computer system while ostensibly performing some innocuous function.
  • Software that enables a user to obtain covert information about another's computer activities by transmitting data covertly from their hard drive.
  • The fraudulent practice of sending emails purporting to be from reputable companies in order to induce individuals to reveal personal information, such as passwords and credit card numbers.
  • A cyber-attack in which the perpetrator seeks to make a machine or network resource unavailable to its intended users by temporarily or indefinitely disrupting services of a host connected to a network.
Protection
  • Firewall
  • Spam Filter
  • Proxy Server
  • Antivirus Software
  • Cryptography
  • Public Key Encryption
  • A part of a computer system or network which is designed to block unauthorized access while permitting outward communication.
  • A program used to detect unsolicited, unwanted and virus-infected emails and prevent those messages from getting to a user's inbox.
  • A server application that acts as an intermediary between a client requesting a resource and the server providing that resource.
  • A computer program used to prevent, detect, and remove malware.
  • The practice and study of techniques for secure communication in the presence of adversarial behavior. More generally, cryptography is about constructing and analyzing protocols that prevent third parties or the public from reading private messages.
  • The field of cryptographic systems that use pairs of related keys. Each key pair consists of a public key and a corresponding private key.
